# Go Google Scraper
#### Gathering Google search information with your own keywords

## Prerequisite
* [Go - 1.15](https://golang.org/doc/go1.15)


## Usage
#### Run the Go application for development
  ```sh
  make start-dev
  ```
To visit app locally: `localhost:8080`

### Run tests
```sh
make test
```
